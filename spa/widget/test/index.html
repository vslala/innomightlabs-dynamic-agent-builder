InnomightLabs Chat Widget Test

This page demonstrates the embeddable chat widget. Look for the chat bubble in the bottom-right corner.

Note: For local testing, you'll need:
  1. Start the API backend: cd api && uv run uvicorn main:app --reload
  2. Create an agent and generate an API key via the dashboard
  3. Update the apiKey below with your generated key
  4. Serve this test page: cd spa/widget && npx serve test

Usage

<script src="widget.js"></script>
<script>
  InnomightChat.init({
    apiKey: 'pk_live_your_api_key_here',
    position: 'bottom-right',  // or 'bottom-left'
    theme: 'light',            // or 'dark'
    primaryColor: '#6366f1',   // optional custom color
    placeholder: 'Type a message...'
  });
</script>

API Methods

  • InnomightChat.init(config) - Initialize the widget
  • InnomightChat.open() - Open the chat window
  • InnomightChat.close() - Close the chat window
  • InnomightChat.destroy() - Remove the widget

Configuration Options

Option Type Description
apiKey string Required. Your public API key (pk_live_xxx)
apiUrl string API base URL (defaults to production)
position string 'bottom-right' or 'bottom-left'
theme string 'light' or 'dark'
primaryColor string Hex color for buttons and accents
placeholder string Input field placeholder text
